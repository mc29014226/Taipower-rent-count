# Taipower Rent Count

這是一個簡單的台電租屋電費計算工具，已部署於 GitHub Pages。

## 部署技術詳解
- **OS**: Linux x64
- **CI/CD**: Manual Atomic Push via Golem
- **Status**: Stable

## 維護備註
此專案曾在高 Steal Time (30%) 環境下完成部署優化。
